Welcome to CloudBSD Website!

CloudBSD is an open source operating system that aims to be lightweight and easy to use.

Its based on OpenBSD 7.8, which is one of the most used os's for servers.

But CloudBSD has a different approach, it aims to make your homelab a ton better and easy to setup, not for cloud servers

Why choose it?

Most BSD versions require you to be an expert at the terminal to do basic tasks like make/delete files

CloudBSD comes with an interface so easy even an tech noobie can use BSD like it's Windows!
